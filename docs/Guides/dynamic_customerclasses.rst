===================================
How to Set Dynamic Customer Classes
===================================

Ciw has two ways in which customers can change their customer class:

+ :ref:`Probabilistically after service at a node <changeclass-afterservice>`.
+ :ref:`At a randomly distributed time while queueing <changeclass-whilequeueing>`.

These can be used in conjunction to one another.
